Indigo Plateau is a simple ranking creation plugin that can be used in videogames tourneys.

== Installation ==

1. Upload 'indigo-plateau.php' and 'indigo-plateau-admin.php' to the '/wp-content/plugins/' folder;
2. Activate the plugin through the 'Plugins' menu in WordPress;
3. Use the admin menu in WordPress to add or remove ranking entries;
4. Place [indigo_plateau_ranking] in your posts or pages to show the ranking status;
5. Place [indigo_plateau_reasons] in your posts or pages to show the currently valid rules for which they may gain points.
